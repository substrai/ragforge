# substrai-ragforge

**Config-driven enterprise RAG architecture generator for serverless infrastructure.**

> This is the npm placeholder package. The primary implementation is in Python.

## Installation

### Python (primary)

```bash
pip install substrai-ragforge
```

### npm (TypeScript SDK - coming soon)

```bash
npm install substrai-ragforge
```
